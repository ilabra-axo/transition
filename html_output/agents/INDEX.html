    🔧 Implementation Guide

    Prerequisites

    • Anthropic Claude API access (Claude 3.5 Sonnet recommended)
    • Python 3.9+ with anthropic, json, requests libraries
    • Optional: MCP (Model Context Protocol) servers for enhanced capabilities

      • Getting Started (5 minutes)

    • Read the quick reference:
      •    cat QUICK_REFERENCE.md
    • Pick an agent based on your need (see table above)
    • Load the agent card:
      •    import json
   with open('sdv-suite/cards/sensorfusionanalyst.json') as f:
       agent = json.load(f)
    • Execute a query (see sdv-suite/README.md)

      • Full Setup (30 minutes)

      Follow the detailed guide in sdv-suite/README.md
